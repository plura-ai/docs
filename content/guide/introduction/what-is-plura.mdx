## This is Plura ! 

Plura is a cutting-edge SaaS platform that redefines AI-powered communication and interaction. Our innovative solution empowers users to create intelligent "brains" - custom AI agents that can be trained with specific information and connected across multiple communication channels.

## Open Source Project

Plura is an open-source project, allowing for community contributions and customizations. The tech stack includes:

- Frontend: React with Next.js
- Backend: Node.js
- Database: PostgreSQL
- Redis instance (Upstash)

Here is the Plura repository at: [https://github.com/plura-ai/plura/](https://github.com/plura-ai/plura/)
